# 这是什么  
获取弹药的小帮手  

# 为什么  
找弹药太麻烦了不是吗，老是复制一句又不好使，现在直接从搜索超话的结果里获取最新最热的弹药。  

# 怎么用  
配虚拟环境，装依赖，跑 weibo.py。

# Cookies 和 Logging
这些都可以不用配置也不用管。  

# 还有呢

Set up your venv, install one module from requirements.txt, run weibo.py and you are ready to go.
